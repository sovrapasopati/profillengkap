    Mengapa saya tertarik dengan web development

    Saya tertarik dengan web development karena bidang ini memungkinkan saya untuk menggabungkan kreativitas dan logika dalam sebuah karya nyata. Melalui web development, saya bisa menuangkan ide-ide menjadi website yang tidak hanya menarik secara tampilan, tetapi juga fungsional dan bermanfaat bagi orang lain. Selain itu, hasil belajar di bidang ini dapat langsung terlihat dan diuji melalui browser, sehingga membuat proses belajar terasa lebih menyenangkan dan memotivasi. Saya juga melihat bahwa web development memiliki peluang yang luas di masa depan, baik untuk membangun portofolio pribadi, mengembangkan bisnis online, maupun berkarier sebagai developer.
